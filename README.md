# LLM_chat_playground.py
This project is LLM chat playground where you can play with temperature values and get the result from ChatGpt chatmodel.
It has a basic interface where you can choose from three temperature values and get the result from the LLM API call.
The second mode gives you a default value basis the task type selected. There is a slider given as well for you to manually change the value in this and see results.
Finally, there is a comparison feature as well. You can utilize it to see the response of LLM call basis three different temperature values and see how results differ.
